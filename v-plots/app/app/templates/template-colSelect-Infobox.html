Select Columns option for CSV upload

Requirements for Dataset File:

  • Comma separated CSV file.
  • The columns that needs to be selected for generating histograms, must be either integers or decimals.
  • No text is allowed for the columns selected for generating histograms.

Creation of Histogram:

  • After CSV upload, the user needs to select two columns from the drop-down menu for creating histograms. The drop-down menu is generated from the uploaded file.
  • If there are no error with the data, the data from the respective columns are used to generate the histograms.
  • Data from only one column is used for the respective (top/left or bottom/right) histogram.
