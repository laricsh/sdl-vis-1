Select Column and Filter option for CSV upload

Requirements for Dataset File:

  • Comma separated CSV file.
  • The columns that needs to be selected for generating histograms, i.e. the Data Source column must be either integers or decimals.
  • No text is allowed for the columns selected for generating histograms.

Creation of Histogram:

  • After CSV upload, the user needs to select a column as Data source from the drop-down menu for creating histograms. The drop-down menu is generated from the uploaded file.
  • If there are no error with the data, the data from the respective column is used to generate the histograms.
  • For Filter, the user needs to select some other column. This column may contain non-numerical data.
  • User then needs to select a filter value for top/left and bottom/right distribution. The values for these filters are the distinct values from the column selected as filter.
  • The data extracted from the column selected as data source, is then filtered into two groups based on the filter values selected for top/left and bottom/right distributions.
  • The filtered data is then used to generate histograms.

Example Dataset:

Marks Status Country Grade
10 Bachelor Germany 3.0
23 Master Spain 2.3
30 Master Germany 2.0
15 Doctoral USA 2.7
30 Doctoral Spain 2.0

For e.g., for the given data set example, consider that the user selects Marks as the data source column and Status as the filter column. For top/left and bottom/right filter values, Bachelor, Master and Doctoral will be available. Let's say the user selects Bachelor for top/left distribution. Then the data from Marks column will be selected, whose rows have Bachelor in the Status column. In this case it will be: 10 only. Similarly, the data from the Marks column will also be selected for bottom/right distribution filter value.
